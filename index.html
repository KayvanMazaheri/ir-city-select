استفاده

برای استفاده کافی است تکه کد زیر را در فرم خود قرار دهید

<!-- Inside HTML <form> -->
<div class="ir-select">
    <select class="ir-province"></select>
    <select class="ir-city"></select>
</div>

سپس این پروژه را به یکی از روش های زیر به صفحه HTML اضافه کنید
توجه: این پروژه برای اجرا شدن نیاز به jQuery دارد.

۱. استفاده از CDN (توصیه می‌شود)
<!-- jQuery -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.1.0/jquery.min.js"></script>

<!-- ir-city-select.js -->
<script src="https://github.com/KayvanMazaheri/ir-city-select/releases/download/v0.2.0/ir-city-select.min.js"></script>
۲. از Github دانلود کنید
دانلود از Github
۳. دریافت جدید ترین لیست شهر‌ها و ساختن نسخه‌ی خودتان

قدم اول: کد را دانلود کنید

git clone https://github.com/KayvanMazaheri/ir-city-select.git && cd ir-city-select

قدم دوم: کد را تغییر دهید (اختیاری)

vim index.js

قدم سوم: نسخه خودتان را بسازید
نکته: نیاز به gulp دارید

gulp

نسخه‌ی شما آماده است!

ir-city-select/
└── dist/
    ├── ir-city-select.js
    └── ir city-select.min.js
